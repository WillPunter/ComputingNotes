Translation of Predicate Logic

  • Translation of quantifiers - \(\forall\) translates to "for all" or "every". \(\exists\) translates to "there exists" or "for some".
  • Translation of variables - variables should not be used in the English translation of a formula. Rather than using the variable names, we should use the word for the object they represent.
  • Examples:
    • "All lecturers are human" \(\equiv \forall x (Lecturer(x) \rightarrow Human(x))\)
    • "Some lecturer is human" \(\equiv \exists x (Lecturer(x) \wedge Human(x))\)
    • "Frank bought a PC" \(\equiv \exists x (PC(x) \wedge Bought(Frank, x))\)
