Propositional Logic Semantics

Definitions

  • Situation - a situation in logic is something that determines truth values. In propositional logic, a situation is sim[ly an assignment of true (\(tt\)) and false (\(ff\)) values.
  • Atomic evaluation function - an atomic evaluation function is a situation in propositional logic. If \(A\) is a set of propositional atoms, an atomic evaluation function will be of the form \(u : A \rightarrow {tt, ff}\), i.e. it will map each propositional atom to either \(tt\) or \(ff\).
  • Evauation function - if \(A\) is a set of propositional atoms and \(v\) is an atomic evaluation function for \(A\) then the evaluation function \(|...|_v\) assigns the truth values \(tt\) and \(ff\) to a formula by following these rules:
    • If \(\phi \in A\) then \(|\phi| = v(\phi)\), i.e. \(|\phi| = tt\) if and only if \(v(\phi) = tt\).
    • \(|¬\phi|_v = tt iff |\phi|_v = ff\).
    • \(|\phi \wedge \psi|_v = tt iff |\phi|_v = tt\) and \(|\psi|_v = tt\).
    • \(|\phi \vee \psi|_v = tt iff |\phi|_v = tt\) or \(|\psi|_v = tt\).
    • \(|\phi \rightarrow \psi|_v = tt iff |¬\phi|_v = tt\) or \(|\psi|_v = tt\).
    • \(|\phi \leftrightarrow \psi|_v = tt iff |\phi|_v = |\psi|_v\).
    • \(|\bot|_v = ff\).
    • \(|\top| = tt\).
  • Truth functionality - Boolean connectives are truth function in an evaluation function - this means that the truth-value of a formula is functionally determined by the truth-values of the connected subformulas.
  • Truth tables - truth tables are a way of evaluating logical formulas by considering the possible values of each propositional atom and evaluating the rest of the formula. The truth tables for the propositional Boolean connectives are shown below. Each row of a truth table represents one situation.

  • Truth table connectives - any truth table can defined a connective, although not all of them are given names. The Sheffer stroke, denoted \(\uparrow\) is similar to NAND in electronics. It has the following truth table:
    \(\phi\)\(\psi\)\(\phi \uparrow \psi\)
    ttttff
    ttfftt
    fftttt
    fffftt
  • Functional completeness - a set of Boolean connecives, \(C\) is functionally complete if any connective of any arity can be defined just in terms of the connectives in \(C\).
    • E.g. the set \(\{\uparrow\}\) is functionally complete.
    • E.g. the set \(\{¬, \wedge\}\) is functionally complete.
