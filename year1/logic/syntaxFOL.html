Predicate Logic Syntax

  • There are six syntactic features of predicate logic:
    1. Predicates (these take objects arguments and return a true or false value)
    2. Constants (identifiers assigned to objects)
    3. Variables (identifiers which can take the value of any object)
    4. Quantifiers (for all \(\forall\) and there exists \(\exists\))
    5. Functions (these take arguments and return objects)
    6. Equality
  • Objects are simply jsut elements defined in a predicate logic.
  • Predicate symbols - predicate symbols describe properties of and relations between objects.
  • Constants - names assigned to objects
  • Quantifiers - the quantifiers in predicate logic are for all \(\forall\) and there exists \(\exists\). E.g. if we wanted to write that all objects are students (i.e. satisfy the student predicate) we would write \(\forall x student(x)\). "Frank has a student friend" can be written as \(\exists x(student(x) \wedge friend(frank, x))\).
  • Functions - functions take objects as arguments and return objects.
  • Equality - two symbols are the same if they refer to the same objects.
  • Signature - the signature of a predicate logic, is a triple \(\langle K, F, P \rangle\) where \(K\) is a (possibly empty) set of constants, \(F\) is a possibly empty set of function symbols and \(P\) is a set of predicate symbols.
  • Term - a term is:
    1. Any constant in \(L\) is an \(L\)-term.
    2. Any variable is an \(L\)-term.
    3. If \(f\) is an \(n\)-ary function symbol in \(L\) and \(\t_1, ..., t_n) are \(L-terms\) then f(t_1, ..., t_n) is an \(L-term\).
    4. Nothing else is an \(L-terms\).
  • A closed term, also called a ground term, is one that doesn't involve a variable.
  • Formula - a formula is:
    1. If \(R\) is an \(n\)-ary predicate symbol in \(L\), and \(t_1, ..., t_n\) then \(R(t_1, ..., t_n)\) is an atomic \(L\)-formula.
    2. If \(t, t'\) are \(L\)-terms then \(t = t'\) is an atomic \(L-formula\).
    3. \(\top\) and \(\bot\) are atomic \(L\)-formulas.
    4. If \(\phi\) and \(\psi\) are \(L\)-formulas then so are (¬\phi), \(\phi \wedge \psi\), \(\phi \vee \psi\), \(\phi \rightarrow \psi\), \(\phi \leftrightarrow \psi\).
    5. If \(\phi\) is an \(L\)-formula and \(x\) is a variable, then \(\forall x \phi\) and \(\exists x \phi\) are \(L-formulas\).
    6. Nothing else is an \(L-formula\).
  • Difference between a term and a formula - remember that a term evaluates to an object and a formula evaluates to a Boolean value.
  • Atomic formula - an atomic formula is a predicate symbol with arguments filled in with terms, e.g. \(Lecturer(susan)\), \(PC(x)\), etc.
  • Literal - a literal is an atomic formula or its negation.
  • Sentence - a sentence is a formula with all variabls in the scope of a quantifier, e.g. \(\forall x \forall y Bought(x, y)\).
  • \(\forall x\) and \(\exist x\) have the same strength as \(¬\) in the binding conventions.
